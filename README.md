## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/your-portfolio.git
```

### Navigate to the Project

```bash
cd your-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
